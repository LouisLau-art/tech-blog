Onu UI 是一个基于 UnoCSS 的**原子化** UI 库，它**不依赖于特定框架**。

*   **核心实现:** 所有组件都是通过 **UnoCSS 快捷方式**实现的纯原子化 CSS 库。
*   **性能:** UnoCSS 会在匹配过程中缓存并最小化压缩，性能极高。
*   **主题与颜色:** Onu UI 支持**动态主题**，通过 [Magicolor](https://github.com/zyyv/magic-color) 生成符合人类视觉感知的一整套颜色系统,。默认主题色是**随机的**，可以通过 `color` 选项配置主题色。
*   **预设集成:** `@onu-ui/preset` 内部集成了常用的 UnoCSS 预设，提供一站式解决方案：
    *   `presetUno()`: UnoCSS 默认预设。
    *   `presetAttributify()`: 属性化模式。
    *   `presetIcons()`: 图标预设。
    *   `presetWebFonts()`: Web 字体预设。
*   **字体:** 默认使用 **DM Sans** 字体，可通过 `font-onu` 工具类应用。

### Radio (单选框)
**描述:** 允许用户从一组选项中选择单个选项的表单控件。
| 配置类别 | UnoCSS 快捷方式/配置 | 描述 |
| :--- | :--- | :--- |
| **基础用法** | - | 基础单选框,。 |
| **颜色** | `radio-<color>` | 更改单选框的颜色。 |
| **大小** | `radio-<size>` | 更改单选框的大小。 |
| **禁用** | `disabled` | 禁用单选框。 |
